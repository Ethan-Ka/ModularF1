# Season Assets

This directory contains season-specific static assets that ship with Pitwall.

## Structure

- `2026/cars/shapes/`: Car SVG shape assets by team.
- `2026/tracks/layouts/`: Track layout assets and metadata.
- `2026/drivers/headshots/`: Driver headshots shipped locally.
- `2026/teams/logos/`: Team logos.
- `2026/teams/colors.json`: Team color palettes (including secondary accents).
- `2026/manifest.json`: Season asset manifest and versioning.

Future seasons should be added as sibling folders (`2027`, `2028`, ...), each with the same structure.
